# Profile
Profile Card Component
A fully accessible, responsive, and semantically rich Profile Card built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies.

Preview

A clean, responsive card displaying a user's avatar, name, biography, social links, hobbies, dislikes, and live epoch time in milliseconds.

Features

✅ Semantic HTML5 (<article>, <figure>, <nav>, <section>, <header>)
✅ WCAG AA accessible — color contrast, focus styles, alt text, aria-live
✅ Fully keyboard-navigable (Tab through all links, visible focus rings)
✅ Live epoch timestamp (Date.now()) updated every 1000ms
✅ Responsive layout — stacks vertically on mobile, side-by-side on wider screens
✅ All elements include data-testid attributes for automated testing
✅ Social links open in new tab with rel="noopener noreferrer"
✅ Zero dependencies — plain HTML, CSS, JS
